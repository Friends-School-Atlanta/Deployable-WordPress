Easily add a notification bar to your Genesis child themes.

== Description ==

This plugin allows Genesis child themes to easily display notifications to their users. Using the a few of Genesis' hooks, administrators can choose the notifications to display above or below the sites header. This plugin allows for a few settings such as giving the notification a specific background color and allowing to limit the notification to just the homepage. 

== Installation ==


1. Upload `genesis-scrolling-notification` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. With your new "Notifications" option in the admin bar, simply add a new notification.
4. Adjust the settings to your prefrence.

== Frequently Asked Questions ==

= Will this work without the Genesis Framework =

No, this plugin uses Genesis specific hooks to display the notifications
